API Documentation | Nataraj AI

Introduction

This document provides an overview of the available API routes, their methods, and the expected input and output formats for Nataraj AI. If you were intended to use the application, please head to https://nataraj-ai.web.app.

1. Get All Items

Route: /api/items

Method: GET

Description: Fetches a list of all items.

Response: A JSON array of item objects.


Example Response:
[
    {
        "id": 1,
        "name": "Item 1",
        "description": "Description for item 1"
    },
    {
        "id": 2,
        "name": "Item 2",
        "description": "Description for item 2"
    }
]

2. Get Item by ID

Route: /api/items/{id}

Method: GET

Description: Fetches a single item by its ID.

URL Parameters: id (integer) - The ID of the item to fetch.

Response: A JSON object of the item.


Example Response:
{
    "id": 1,
    "name": "Item 1",
    "description": "Description for item 1"
}

3. Create a New Item

Route: /api/items

Method: POST

Description: Creates a new item.

Request Body: A JSON object containing the new item's details.


Example Request Body:
{
    "name": "New Item",
    "description": "Description for new item"
}

Response: A JSON object of the created item.


Example Response:
{
    "id": 3,
    "name": "New Item",
    "description": "Description for new item"
}

4. Update an Item

Route: /api/items/{id}

Method: PUT

Description: Updates an existing item by its ID.

URL Parameters: id (integer) - The ID of the item to update.

Request Body: A JSON object containing the updated item's details.


Example Request Body:
{
    "name": "Updated Item",
    "description": "Updated description for item"
}

Response: A JSON object of the updated item.


Example Response:
{
    "id": 1,
    "name": "Updated Item",
    "description": "Updated description for item"
}

5. Delete an Item

Route: /api/items/{id}

Method: DELETE

Description: Deletes an item by its ID.

URL Parameters: id (integer) - The ID of the item to delete.

Response: A success message.


Example Response:
{
    "message": "Item deleted successfully"
}
